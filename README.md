# Hackathon Finder 🚀

Hackathon Finder is a web application that helps students discover hackathons happening across India and nearby colleges.

## Features
- Search hackathons by state, city, or college
- Displays hackathon locations and details
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
